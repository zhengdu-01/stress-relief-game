# 解压小游戏 - iOS

一个采用星露谷风格卡通像素画风的iOS解压小游戏。

## 功能特点

### 核心玩法
- **普通攻击（狂戳）**：快速点击领导角色造成少量伤害，同时积累怒气值
- **必杀技（拖拽）**：怒气值满时，随机出现武器卡牌，拖拽到领导身上释放造成大量伤害

### 角色系统
- 可选择男领导或女领导
- 自定义输入领导名字
- 4种表情变化（傲慢→愤怒→委屈→求饶）
- 眩晕特效（使用Tiffany饰品时）

### 武器系统
- 烂菜叶：基础武器，伤害10-15
- 臭鸡蛋：基础武器，伤害10-15
- 菜刀：特殊武器，伤害30
- Tiffany饰品：传奇武器，伤害50，附带眩晕效果

### 游戏场景
- 卡通像素风办公室场景
- 办公桌、电脑、文件柜、窗户等元素
- 星露谷风格配色

### 胜利条件
- 60秒内将领导血量打空即可获胜

## 技术栈

- SwiftUI
- iOS 17.0+
- 原生Swift开发

## 运行项目

1. 使用Xcode 15+打开 `解压小游戏.xcodeproj`
2. 选择目标设备（iPhone模拟器或真机）
3. 点击运行按钮或按Cmd+R

## 游戏玩法说明

1. 启动游戏后，选择领导性别并输入名字
2. 点击"开始游戏"进入战斗
3. 快速点击领导进行普通攻击
4. 当怒气槽满时，顶部会出现武器卡牌
5. 将武器卡牌拖到领导身上释放必杀技
6. 在60秒内打空血量即可获胜
